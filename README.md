# Showcase

This repository contains my personal **showcase landing page**, built to showcase my experience as a **Senior React / Frontend Developer**.

The goal of this project is not just presentation, but demonstrating:
- strong architectural thinking
- clean separation of concerns
- scalability and extensibility
- ability to blend **engineering with design**

---

## ✨ Concept

The landing page follows a **minimalist, high-contrast design**:
- black & white base
- subtle **orange accent** for focus and interaction
- lightweight visual layer (background + graphics)
- responsive across mobile and desktop

The design aims to feel:
- modern
- intentional
- slightly experimental, but controlled

---

## 🧠 Approach

This is not a template-driven portfolio.

It is structured as a **modular frontend system**, where:
- UI is separated from logic
- reusable patterns are prioritized
- future features (like game prototype) can be added without refactoring

The project reflects how I approach real-world applications:
- scalable structure
- composable components
- predictable state handling

---

## 🧩 Features

- Section-based layout (Hero, About, Experience, Skills, Contact)
- Custom animated background (PixiJS-ready architecture)
- Placeholder for **interactive game prototype**
- Responsive design with breakpoint-driven behavior
- Clean typography and spacing system
- Subtle motion and visual feedback

---

## 🏗️ Project Structure

```txt
src/
  assets/           # static images
  claude/           # prompt / design context
  components/       # reusable UI components
    background/
    hero-background/
    layout/
    ui/
  data/             # content definitions
  hooks/            # custom React hooks
  sections/         # page sections
  styles/           # global + variables
  utility/          # helpers
  ```
